講題：SafeSlinger: easy-to-use and secure public-key exchange

SafeSlinger: easy-to-use and secure public-key exchange

SafeSlinger is a system for secure exchange of authentic information, such as secure messaging and email, using smartphones while preserving user privacy without any external trusted parties. As an ongoing research project at Carnegie Mellon University’s CyLab, we have recently released an open-source platform for SafeSlinger public-key and contact information exchange, rich text-messaging, and secure friend introduction applications, and a secure email application is currently under development, all of which are maintained for a variety of Android and iOS devices and up-to-date OS versions.

Our central goal is to develop a system that provides strong security guarantees with usability in mind. We also want to support flexibility and interoperability such that users with different OS vendors can still use SafeSlinger remotely (even across countries). However, a core challenge is to resolve the tension among security, usability, and flexibility. To support flexibility, we use Internet-based communication such that all devices connect to a cloud server using IP; however, such a decision indicates that users are vulnerable to privacy breaches with the compromised server or any man-in-the-middle attackers. Consequently, we address authenticity, secrecy, and privacy by constructing a multi- commitment hash tree along with efficient Diffie-Hellman secret sharing to ensure that users’ private information is shared only with the intended recipients if and only if every step in the exchange protocol runs successfully.

講題：環評為何這麼容易被判決撤銷？－兼談經濟與環保的拔河

環評為何這麼容易被判決撤銷？－兼談經濟與環保的拔河

依環評法第4條第2款的定義，環評是事前對開發行為的客觀、科學、綜合的調查、預測、分析及評定其對環境影響的範圍及程度。因此「事前對環境現況的客觀、科學調查」，攸關對於評估開發行為究竟會造成如何影響的正確性，至為重要。為什麼環評容易被法院判決撤銷？主要原因即在顧問公司及其技師於環境現況調查過程中，為刻意隱匿對其不利的資訊，而在調查方法（包括時間、地點、路徑），或對環境現況的描述方式上取巧。再加上政府經濟至上的迷思，浮濫通過，反而適得其反。本次講題嚐試揭露其中內幕、並反思如何確保環境資訊與行政程序的公開、公正，使經濟發展與環境保護得以兼籌並顧？以利促進人民有真正好的生活環境。
